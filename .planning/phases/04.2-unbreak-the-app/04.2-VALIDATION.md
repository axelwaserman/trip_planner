| **Full suite command** | `just test` (excludes slow/e2e) |

| **Estimated runtime — unit (be+fe)** | ~25–40 seconds |
| **Estimated runtime — full default suite** | ~60–90 seconds |

---

## Sampling Rate

- **After every task commit:** Run the closest-scoped unit suite for the file changed (e.g. `cd backend && uv run pytest tests/unit/test_auth_probe.py -v`).
- **After every plan wave:** Run `just test` (backend default) and `cd frontend && npm test -- --run` (frontend).
- **Before `/gsd-verify-work`:** `just check` + `just test` must both be green; manual UAT on the golden path.
- **Max feedback latency:** ≤ 90 seconds for the full default suite.
