# Plan 04.2-02 Summary — be-auth-and-probe

## Outcome

Renamed the auth router mount to `/api/auth`, fixing the Vite proxy gap (RESEARCH.md
Pitfall 1) where the frontend's `/api/*` proxy could never reach `/token` or `/users/me`.
Added `GET /api/auth/me` (formerly `/users/me`, renamed per D-17). Created the
`provider_probe` service module with `probe_provider()` async function and the `ProbeError`
Pydantic model that classifies httpx connect/timeout/HTTP errors into structured
`ollama_unreachable` / `model_not_installed` / `missing_api_key` errors. Added
`SessionCreateError` to `models.py` for use by Plan 04.
